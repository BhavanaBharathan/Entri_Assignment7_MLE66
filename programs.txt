1. Write a program to count the number of digits in a given number. Example: Input: num = 98765 Output: 5

 2.Write a program to find the product of all digits in a number. Example: Input: num = 234 Output: 24 (2 × 3 × 4) 
 
 3. Write a program that counts the number of words in a given sentence. Example: Input: "Hello world, how are you?" Output: 5 words 
 
 4. Write a program that reverses the order of words in a sentence but keeps the words intact.
 Example: Input: "JavaScript is fun" Output: "fun is JavaScript" 
 
 5. Write a program to check if a given sentence is a pangram (contains every letter from A-Z). 
 
 Example: Input: "The quick brown fox jumps over the lazy dog" Output: "It is a pangram"
